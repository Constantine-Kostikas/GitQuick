# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Refresh MR list with `r` key

## [0.1.3] - 2026-01-25

### Fixed

- GoReleaser v2 configuration compatibility

## [0.1.2] - 2026-01-25

### Fixed

- GoReleaser build configuration

## [0.1.1] - 2026-01-25

### Added

- GPG signing for releases

## [0.1.0] - 2026-01-25

### Added

- Initial release of gtui
- MR/PR browsing with interactive TUI
- Author filtering with `@me` quick reference
- One-command branch checkout with automatic fetch/pull
- Platform auto-detection for GitHub and GitLab
- Interactive tab-based dashboard (MRs, Issues, Branches)
- Real-time checkout progress with step-by-step feedback
- Keyboard navigation (arrow keys, enter, tab)
- GoReleaser configuration for multi-platform builds
- GitHub Actions CI/CD workflows
